# Niewydolność serca — prezentacja (ESC)

Aplikacja Next.js z krótką prezentacją wytycznych ESC dotyczących niewydolności serca (2021 + aktualizacje 2023).

## Uruchomienie lokalne

```bash
npm install
npm run build
npm start
```

Otwórz http://localhost:3000.

- Nawigacja: strzałki ← →, spacja, PageUp/PageDown, Home/End.
- Druk/PDF: przycisk "Drukuj / PDF" w nagłówku.

## Źródła
- ESC Guidelines for the diagnosis and treatment of acute and chronic heart failure (2021)
- 2023 Focused Update for HFrEF therapy / SGLT2 in HFmrEF/HFpEF

Uwaga: Materiał edukacyjny. Sprawdź najnowsze dokumenty ESC dla pełnych szczegółów i klas zaleceń.
